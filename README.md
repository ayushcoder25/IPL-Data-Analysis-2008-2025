# 📊 IPL Data Analysis (2008–2025)

## 🚀 Overview
This project analyzes IPL data from 2008 to 2025 using Python, Pandas, and Matplotlib.  
The goal is to extract insights related to team performance, player performance, and match strategies.

---

## 🧹 Data Cleaning
- Merged datasets from different sources
- Standardized column names
- Fixed inconsistent values (team names, toss decisions)
- Handled missing values
- Converted date formats

---

## 📈 Analysis Performed
- Most Wins by Teams
- Top Players by Player of the Match Awards
- Toss Impact Analysis
- Venue-Based Analysis
- Team Dominance at Specific Venues

---

## 💡 Key Insights
- Teams choosing to field after winning toss have higher success rate
- Toss impact varies across venues
- Some teams dominate specific stadiums
- Top players show consistent performance across seasons

---

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

---

## 📌 Conclusion
This project highlights how data analysis can uncover strategic insights in sports analytics.
